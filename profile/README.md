## Overview

Yate Advanced is a professional music tag editor for organizing audio file metadata including titles, artists, albums, artwork, lyrics, and dozens of other tag fields across MP3, FLAC, AAC, AIFF, WAV, and other audio formats. It provides the tag editing depth that music collectors and library managers need to maintain consistently organized music collections with accurate, complete metadata across thousands of files.

Tag editing handles ID3v1, ID3v2, Vorbis Comments, APE, and other tag formats with support for every standard and custom field that audio metadata can contain. The editor displays all tag fields in a spreadsheet-like interface where batch editing applies changes across multiple files simultaneously. Field mapping handles the differences between tag formats transparently.

Online lookup retrieves tag information, artwork, and lyrics from MusicBrainz, Discogs, and other databases by matching audio fingerprints or existing partial metadata. The lookup fills in missing information and corrects errors without manual data entry. Artwork retrieval downloads high-resolution album covers that replace missing or low-quality embedded images.

Batch processing applies tag operations — renaming, field manipulation, artwork embedding, format conversion — across entire albums, artists, or libraries with configurable action sequences. The batch engine handles the volume processing that organizing large music collections requires. File renaming constructs filenames from tag fields using configurable patterns.

For music collectors and library managers who need precise control over audio metadata with the editing depth and batch capability that maintaining large music collections demands, Yate Advanced is the tag editor that handles every metadata challenge.

## Performance

Yate balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
